# 🌱 AI Crop Advisor

## 📌 Overview
AI Crop Advisor is a full-stack web application that provides intelligent crop recommendations based on user inputs such as soil type, location, and environmental conditions.

The goal of this project is to assist users in making informed agricultural decisions using data-driven insights.

---

## 🚀 Features
- Smart crop recommendation system
- Interactive and responsive UI
- Real-time user input processing
- API-based architecture
- Clean and scalable code structure

---

## 🛠️ Tech Stack

### Frontend:
- React.js (Vite)
- Tailwind CSS

### Backend:
- Node.js
- Express.js

### Other:
- REST APIs
- Environment-based configuration
